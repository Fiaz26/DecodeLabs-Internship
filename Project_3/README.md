## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose

## Features

- Database Schema Design
- CRUD Operations
- MongoDB Integration
- API Endpoints
- Data Validation
- Error Handling

---

## Installation

```bash
npm install
```

---

## Run Project

```bash
npm run server
```

---

## API Endpoints

### Create Task
POST /api/tasks

### Get All Tasks
GET /api/tasks

### Get Single Task
GET /api/tasks/:id

### Update Task
PUT /api/tasks/:id

### Delete Task
DELETE /api/tasks/:id
